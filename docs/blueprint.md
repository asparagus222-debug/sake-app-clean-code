# **App Name**: 清酒品鑑筆記 (Sake Appreciation Notes)

## Core Features:

- 建立與編輯清酒品飲筆記: 首先必須點選連結手機相機或是圖片庫，選擇並上傳1-2張相片，接著提供下拉選單讓使用者輸入清酒的銘柄（品牌名稱）、所屬酒造（釀酒廠）、接著用評分bar(1-5分)選擇甘、酸、苦、旨、澀的程度，甘的部分在bar的前到後顯示辛口/微辛/中口/甘口/大甘口，酸則顯示不酸/微酸/平衡/很酸/超酸，苦則顯示不苦/微苦/平衡/明顯苦味/超苦，旨則顯示無/微旨/平衡/偏長/長，澀則顯示不澀/微澀/平衡/有點澀/很澀，並透過雷達圖顯示分數，最後再給一個評論欄自由填寫品飲描述（如香氣、口感、餘韻、精米步合等），最後提供綜合評分bar(1-10分），提出後可以先預覽頁面，然後再確認送出。
- 瀏覽所有清酒筆記: 以清晰、易讀的列表或卡片形式展示所有儲存的清酒品飲筆記摘要，方便快速閱覽。
- 搜尋與篩選清酒筆記: 讓使用者能夠根據清酒類型、酒造、銘柄（品牌名稱）、關鍵字、日期範圍或評分來搜尋和篩選筆記。
- 智慧清酒品飲摘要工具: 利用AI工具分析使用者輸入的詳細清酒品飲描述、精米步合和評分，自動生成精煉且富有洞察力的清酒品飲體驗總結。
- 資料儲存: 安全地儲存並管理所有清酒品飲筆記數據，確保資料的持久性和可取回性。

## Style Guidelines:

- 背景色：淺米褐 (#F7F2EE) 營造溫暖、自然的筆記本氛圍。
- 主色調：深磚橘 (#954A1A) 象徵品味與細緻的風味，用於主要互動元素與重要標題。
- 輔助色：成熟紅 (#B82E2E) 提供強烈對比，適用於重點提示或操作按鈕。
- 標題字體：'Playfair' (serif 字體)，帶有典雅和精緻感，適合標題和少量文本。
- 內文字體：'PT Sans' (sans-serif 字體)，清晰且人性化，適合長篇閱讀和品飲筆記詳情。
- 使用簡潔、抽象的圖示，微妙地呼應清酒主題（例如，清酒杯、酒瓶、稻穗、酒桶），避免過於具象的設計。
- 佈局應乾淨、有條理，靈感來自精裝筆記本，注重內容層次和易讀性，使用卡片式設計來展示每個品飲筆記。
- 設計平滑、流暢的過渡動畫，例如在添加新筆記或切換不同筆記視圖時，以增強使用者體驗的沉浸感。
